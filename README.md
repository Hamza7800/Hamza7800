## 🚀 Featured Projects

### [Market Flow](https://marketflowapp.vercel.app/) — Multi-Vendor Marketplace

 Full-stack marketplace where independent vendors manage their own storefronts, products, and payouts under one platform.

- **Stripe Connect** — automatic multi-vendor payment splitting via webhooks
- **Optimistic updates** — cart mutations with React Query rollback on failure
- **Smart image diffing** — UploadThing file key identity to avoid unnecessary deletions
- **Dual-layer caching** — `unstable_cache` + React Query invalidated in sync on mutation

`Next.js` `TypeScript` `Drizzle ORM` `PostgreSQL` `Stripe Connect` `React Query` `Better Auth` `UploadThing` `nuqs`

---

### [Flow Desk](https://flowdesk.vercel.app/) — Project Management Tool

Linear-inspired PM tool with org workspaces, drag-and-drop boards, and role-based access control.

- **10–20× navigation speedup** — from 600–1100ms down to ~50ms via parallel DB queries + `HydrationBoundary`
- **dnd-kit** — drag and drop across board and list views with separate mouse/touch sensors
- **RBAC** — owner / admin / member roles scoped per org, enforced in both UI and server actions
- **Org switcher** — auth client → cache bust → `router.replace` + `router.refresh()` sequence

`Next.js` `TypeScript` `Drizzle ORM` `Turso (SQLite)` `Better Auth` `React Query` `dnd-kit`
